To-Do List Manager

A simple command-line to-do list application that allows users to **add, view, and remove tasks.**

#### Features

* Add new tasks
* View all tasks
* Remove tasks by number
* Input validation for safer task removal

#### Technologies

* Python

#### File

```
todolist.py
```

#### Run the Program

```bash
python todolist.py
```
